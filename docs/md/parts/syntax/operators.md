### Basic operation <span id="BasicOperation"></span>

Use simple or complicated math operations easily in the ils-shell or directly through a ils file

```
5 + 5
```

| Usage | Code | Example | Result | 
|--|--|--|--| 
| Plus | + | 5 + 5 | 10 | 
| Minus | - | 5 - 5 | 0 | 
| Multiply| * | 5 * 5 | 25 | 
| Divided| / | 5 / 5 | 1 | 
| Power by | ^ | 5 ^ 5 | 3125 | 
| Modulo | % | 15 % 6 | 3 |
